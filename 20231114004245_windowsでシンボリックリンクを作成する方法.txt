windowsでシンボリックリンクを作成する方法


Windowsでシンボリックリンク（シムリンク）を作成するには、コマンドプロンプトかPowerShellを使用して`mklink`コマンドを利用します。これを行うには、管理者権限でコマンドプロンプトまたはPowerShellを開く必要があります。

以下は、コマンドプロンプトまたはPowerShellでシンボリックリンクを作成する方法です。

1. 管理者権限でコマンドプロンプトまたはPowerShellを開く
 - スタートメニューを開き、「cmd」と検索し、右クリックして「管理者として実行」を選択する。
 - または、スタートメニューで「Windows PowerShell」と検索し、同様に「管理者として実行」を選択する。

2. `mklink`コマンドを使用してシンボリックリンクを作成する
 - ファイルのシンボリックリンクを作成する:
 ```
 mklink Link Target
 ```
 ここで、`Link`は作成するシンボリックリンクのパス、`Target`はリンクの対象となるファイルのパスです。

 - ディレクトリ（フォルダ）のシンボリックリンクを作成する:
 ```
 mklink /D Link Target
 ```
 `/D`オプションを使用してディレクトリ向けのシンボリックリンクを作成します。

例えば、`D:\MySymlink`に`C:\MyFolder`へのシンボリックリンクを作成する場合、次のようにコマンドを実行します。

```
mklink /D D:\MySymlink C:\MyFolder
```

コマンドが成功すると、以下のようなメッセージが表示されます。

```
symbolic link created for D:\MySymlink <<===>> C:\MyFolder
```

また、ハードリンクやジャンクションを作成するオプションもあります：
- ファイルのハードリンクを作成するには `/H` オプションを使用します。
- ディレクトリのジャンクションを作成するには `/J` オプションを使用します。

これらのコマンドはファイルシステムがNTFSであることを要求します。他のファイルシステム、例えばFAT32やexFATではシンボリックリンクは利用できません。また、シンボリックリンクは一般的にパフォーマンスに影響を与えることはなく、アプリケーションがリンク先のファイルやフォルダを直接参照しているように動作します。
