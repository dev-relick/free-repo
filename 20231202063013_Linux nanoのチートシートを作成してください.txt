Linux nanoのチートシートを作成してください


Linuxのnanoは、コマンドラインインターフェースにおける人気のあるテキストエディタの一つです。操作がシンプルで初心者にも優しいため、多くのユーザーに選ばれています。以下にnanoの基本的な操作とショートカットをまとめたチートシートを作成します。

---

**nanoの起動と基本的な操作**
- ファイルを開く: `nano filename`
- 新しいファイル作成: `nano`
- ファイルとともにnanoを起動して特定の行へ移動: `nano +行番号 filename`

**ファイルの読み込み・保存・終了**
- ファイルを保存: `Ctrl` + `O`
- 保存せずに終了: `Ctrl` + `X`
- 行番号、場所、状態を表示: `Ctrl` + `C`

**テキスト編集**
- カット (カーソル位置のテキストを切り取り): `Ctrl` + `K`
- アンカット (カットしたテキストを貼り付け): `Ctrl` + `U`
- カーソル位置の文字を削除: `Ctrl` + `D`
- カーソル位置の単語を削除: `Ctrl` + `T`

**カーソル移動**
- ファイルの先頭に移動: `Ctrl` + `Y`
- ファイルの末尾に移動: `Ctrl` + `V`
- 次のワードへ移動: `Ctrl` + `Space`
- 前のワードへ移動: `Alt` + `Space`

**検索と置換**
- テキストを検索: `Ctrl` + `W`
- 次の検索結果へ移動: `Alt` + `W`
- テキストを置換: `Ctrl` + `\`

**設定と特殊な操作**
- ヘルプ表示: `Ctrl` + `G`
- 行番号表示: `Alt` + `Shift` + `-` (`Alt` + `-`と押すことが多い)
- ソフトラップモードの切り替え: `Alt` + `$`
- 自動インデントモードの切り替え: `Alt` + `}`

**その他の便利なショートカット**
- カーソル行をスクロールアップ/ダウン: `Alt` + `Y` / `Alt` + `V`
- タブをスペースに変換: `Ctrl` + `Q` + `Tab`
- カーソル位置を中央にする: `Ctrl` + `L`
- コメント/コメント解除（設定ファイルなどで便利）: `Alt` + `3`

---

nanoはバージョンによっていくつかのショートカットや機能が異なることもあるため、最新の情報はnanoの公式ドキュメントや`nano --help`コマンドで確認するのがベストです。また、環境によっては`Ctrl`や`Alt`が異なるキーに割り当てられている場合があるため、ローカルの設定も確認してください。
