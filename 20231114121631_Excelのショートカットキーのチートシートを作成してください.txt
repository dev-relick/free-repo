Excelのショートカットキーのチートシートを作成してください


Excelのショートカットキーは非常に多いため、すべてをカバーするのは難しいですが、よく使われる一般的なショートカットキーのチートシートを以下に示します。これらのショートカットキーは、Excelのバージョンによっては異なる場合がありますが、一般的には以下のようになるでしょう。

### 基本的な操作

| ショートカットキー | 機能 |
| :-----------------: | :--- |
| Ctrl + N | 新規ブックを開く |
| Ctrl + O | 既存のブックを開く |
| Ctrl + S | ブックを保存 |
| Ctrl + P | ドキュメントを印刷 |
| Ctrl + F2 | 印刷プレビューを表示 |
| Ctrl + C | 選択したセルをコピー |
| Ctrl + X | 選択したセルを切り取り |
| Ctrl + V | コピーまたは切り取ったセルを貼り付け |
| Ctrl + Z | 操作を取り消す |
| Ctrl + Y | 取り消した操作をやり直す |
| Ctrl + A | 全てを選択 |
| Ctrl + F | 検索 |
| Ctrl + H | 置換 |
| Ctrl + G | 名前でセルを選択 |
| F2 | セル編集モードに入る |
| Esc | セルの編集を取り消す |
| Enter | セルの編集を確定して下のセルに移動 |
| Shift + Enter | セルの編集を確定して上のセルに移動 |
| Ctrl + Shift + L | フィルターの切り替え |

### シート操作

| ショートカットキー | 機能 |
| :---------------------: | :--- |
| Ctrl + Page Down | 次のシートに移動 |
| Ctrl + Page Up | 前のシートに移動 |
| Alt + E, L (または Alt + Shift + F1) | 新しいシートを挿入 |
| Ctrl + K | ハイパーリンクを挿入 |

### セルの選択と移動

| ショートカットキー | 機能 |
| :---------------------: | :--- |
| Ctrl + 矢印キー | 方向キーに沿ってデータの最後に移動 |
| Shift + 矢印キー | セルを範囲選択 |
| Ctrl + Shift + 矢印キー | セルをデータの最後まで範囲選択 |
| Ctrl + Space | 現在の列を選択 |
| Shift + Space | 現在の行を選択 |
| Home | 行の先頭のセルに移動 |
| Ctrl + Home | シートの先頭に移動 |
| Ctrl + End | シートの最後にデータがあるセルに移動 |

### 編集と整形

| ショートカットキー | 機能 |
| :---------------------: | :--- |
| Ctrl + B | ボールド（太字） |
| Ctrl + I | イタリック（斜体） |
| Ctrl + U | アンダーライン（下線） |
| Ctrl + 1 | セルの書式設定ダイアログボックスを開く |
| Ctrl + Shift + "$" | 通貨の書式設定 |
| Ctrl + Shift + "%" | パーセントの書式設定 |
| Ctrl + Shift + "#" | 日付の書式設定 |
| Ctrl + Shift + "@" | 時刻の書式設定 |
| Ctrl + Shift + "!" | 数値の書式設定 |
| Alt + E, S, V | 特殊貼り付け |

### データ操作

| ショートカットキー | 機能 |
| :---------------------: | :--- |
| Alt + = | 範囲を選択し、合計（オートSUM） |
| Ctrl + Shift + "+" | 新しい行または列を挿入 |
| Ctrl + "-" | 行または列を削除 |
| Ctrl + D | 上のセルを下にコピー（下方向にフィル） |
| Ctrl + R | 左のセルを右にコピー（右方向にフィル） |

これらのショートカットキーは一例であり、Excelにはさらに多くの高度なショートカットキーもあります。ヘルプメニューやオンライン資料などを確認して、特定の操作に関するショートカットキーを追加で探してみるとよいでしょう。
