windowsでシンボリックリンクを作成する方法


Windowsでシンボリックリンクを作成するには、コマンドラインツール「mklink」を使用します。これは、Windows Vista以降のバージョンで使用できるツールです。シンボリックリンクを作成するには、管理者権限が必要なコマンドプロンプトを使用する必要があります。

コマンドプロンプトを管理者権限で開くには、スタートメニューで「cmd」と検索、右クリックし「管理者として実行」を選択します。

以下のコマンドを使用して、ファイルまたはディレクトリのシンボリックリンクを作成します。

ファイルのシンボリックリンクを作成する場合:
```
mklink Link Target
```
ここで、`Link`は作成するシンボリックリンクのパス、`Target`はリンクする元のファイルのパスです。

ディレクトリ (フォルダ) のシンボリックリンクを作成する場合:
```
mklink /D Link Target
```
`/D`オプションを追加することで、ディレクトリのシンボリックリンクを作成することを指定しています。

実際の使用例:
```
mklink C:\path\to\link.txt C:\path\to\original.txt
mklink /D C:\path\to\link-folder C:\path\to\original-folder
```

また、ハードリンクやジャンクションといった他形式のリンクを作成することも可能です:

ファイルのハードリンクを作成する場合:
```
mklink /H Link Target
```

ディレクトリのジャンクション (ハードリンクのようなもので古いバージョンのWindowsでも使えます) を作成する場合:
```
mklink /J Link Target
```
