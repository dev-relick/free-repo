vscodeのショートカットのチートシートを作成して


以下はVisual Studio Code (VSCode) のようく使用されるショートカットキーのチートシートです。このチートシートはWindowsとmacOSの両方のショートカットを含んでいます。各ショートカットの前に、Windowsは (Win)、macOSは (Mac) の略語を記載しています。

### 基本操作
- **ファイルを開く**: (Win) Ctrl+O / (Mac) Cmd+O
- **ファイルを保存**: (Win) Ctrl+S / (Mac) Cmd+S
- **ファイルをすべて保存**: (Win) Ctrl+K S / (Mac) Cmd+Option+S
- **新規ファイル**: (Win) Ctrl+N / (Mac) Cmd+N
- **新規ウィンドウ/インスタンス**: (Win) Ctrl+Shift+N / (Mac) Cmd+Shift+N
- **設定を開く**: (Win) Ctrl+, / (Mac) Cmd+,

### 編集操作
- **取り消し**: (Win) Ctrl+Z / (Mac) Cmd+Z
- **やり直し**: (Win) Ctrl+Y / (Mac) Cmd+Shift+Z
- **カット**: (Win) Ctrl+X / (Mac) Cmd+X
- **コピー**: (Win) Ctrl+C / (Mac) Cmd+C
- **貼り付け**: (Win) Ctrl+V / (Mac) Cmd+V
- **検索**: (Win) Ctrl+F / (Mac) Cmd+F
- **置換**: (Win) Ctrl+H / (Mac) Cmd+Option+F
- **行を削除**: (Win) Ctrl+Shift+K / (Mac) Cmd+Shift+K
- **行を上に移動**: (Win) Alt+Up / (Mac) Option+Up
- **行を下に移動**: (Win) Alt+Down / (Mac) Option+Down
- **行のコピーを上に作成**: (Win) Ctrl+Shift+Alt+Up (Mac) Cmd+Shift+Option+Up
- **行のコピーを下に作成**: (Win) Ctrl+Shift+Alt+Down / (Mac) Cmd+Shift+Option+Down

### カーソルと選択範囲
- **単語単位で移動**: (Win) Ctrl+Left/Right / (Mac) Option+Left/Right
- **単語単位で選択**: (Win) Ctrl+Shift+Left/Right / (Mac) Option+Shift+Left/Right
- **行の先頭/末尾に移動**: (Win) Home/End / (Mac) Cmd+Left/Right
- **ドキュメントの先頭/末尾に移動**: (Win) Ctrl+Home/End / (Mac) Cmd+Up/Down
- **ドキュメントの先頭/末尾まで選択**: (Win) Ctrl+Shift+Home/End / (Mac) Cmd+Shift+Up/Down

### ナビゲーションおよび検索
- **コマンドパレット**: (Win) Ctrl+Shift+P / (Mac) Cmd+Shift+P
- **ファイル検索**: (Win) Ctrl+P / (Mac) Cmd+P
- **行に移動**: (Win) Ctrl+G / (Mac) Cmd+G
- **エディタ間を切り替え**: (Win) Ctrl+Tab / (Mac) Ctrl+Tab
- **記号へ移動**: (Win) Ctrl+Shift+O / (Mac) Cmd+Shift+O
- **エラーや警告に移動**: (Win) F8 / (Mac) F8

### コード編集
- **コードの折り畳み/展開**: (Win) Ctrl+Shift+[ / ] / (Mac) Cmd+Option+[ / ]
- **コメントのトグル**: (Win) Ctrl+/ / (Mac) Cmd+/
- **ブロックコメントのトグル**: (Win) Shift+Alt+A / (Mac) Shift+Option+A
- **インテリセンス/自動補完**: (Win) Ctrl+Space / (Mac) Ctrl+Space

### デバッグ
- **デバッグ開始/続行**: (Win) F5 / (Mac) F5
- **デバッグ停止**: (Win) Shift+F5 / (Mac) Shift+F5
- **次のブレークポイントに進む**: (Win) F10 / (Mac) F10
- **関数内をステップイン**: (Win) F11 / (Mac) F11
- **関数からステップアウト**: (Win) Shift+F11 / (Mac) Shift+F11

### その他
- **統合ターミナルを開く**: (Win) Ctrl+` / (Mac) Control+`
- **プレビュー機能（Markdown等）**: (Win) Ctrl+Shift+V / (Mac) Cmd+Shift+V
- **拡張機能ビューを開く**: (Win) Ctrl+Shift+X / (Mac) Cmd+Shift+X
- **ソースコントロールビューを開く**: (Win) Ctrl+Shift+G / (Mac) Cmd+Shift+G
- **問題ビューを開く**: (Win) Ctrl+Shift+M / (Mac) Cmd+Shift+M

これらはVSCodeでの一般的なショートカットですが、カスタマイズで変更が可能です。このリストを参考に、独自のニーズに応じて最も役立つショートカットを網羅的に覚えたり、ドキュメントにまとめたりすることができます。また、VSCodeの公式ドキュメントにアクセスし、最新の情報を確認することも重要です。www.code.visualstudio.com/docs/getstarted/keybindings#_keyboard-shortcuts-reference で公式のキーバインディングドキュメントを参照できます。
