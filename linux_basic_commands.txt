ls - Linuxでディレクトリをリスト表示する最も頻繁に使用されるコマンド
pwd - Linuxで作業中のディレクトリを表示するコマンド
cd - Linuxでディレクトリ間を移動するコマンド
mkdir - Linuxでディレクトリを作成するコマンド
mv - Linuxでファイルを移動または名前を変更する
cp - mvと似ていますが、Linuxでファイルをコピーする際に使用
rm - ファイルやディレクトリを削除する
touch - 空のファイルを作成する
ln - 他のファイルへのシンボリックリンク（ショートカット）を作成
cat - ターミナルにファイルの内容を表示
clear - ターミナルの表示をクリア
echo - コマンドに続く任意のテキストを表示
less - ターミナルでページ出力を表示するLinuxコマンド
man - すべてのLinuxコマンドのマニュアルページにアクセス
uname - OSに関する基本情報を取得するLinuxコマンド
whoami - アクティブなユーザー名を取得
tar - Linuxでファイルを抽出・圧縮するコマンド
grep - 出力内の文字列を検索
head - 上部から指定された行数を返す
tail - 下部から指定された行数を返す
diff - 二つのファイルの違いを見つける
cmp - 二つのファイルが同一かどうかを確認
comm - diffとcmpの機能を組み合わせる
sort - 出力時にファイルの内容を並び替えるLinuxコマンド
export - Linuxで環境変数をエクスポート
zip - Linuxでファイルを圧縮
unzip - Linuxでファイルを解凍
ssh - Linuxでのセキュアシェルコマンド
service - Linuxでサービスを開始・停止するコマンド
ps - アクティブなプロセスを表示
kill and killall - プロセスIDまたは名前でアクティブなプロセスを終了
df - ディスクファイルシステム情報を表示
mount - Linuxでファイルシステムをマウント
chmod - ファイルの権限を変更するコマンド
chown - ファイルやフォルダーの所有権を付与するコマンド
ifconfig - ネットワークインターフェースとIPアドレスを表示
traceroute - 目的地に到達するまでのネットワークホップをトレース
wget - インターネットから直接ファイルをダウンロード
ufw - ファイアウォールコマンド
iptables - 他のすべてのファイアウォールユーティリティがインターフェースする基本ファイアウォール
apt, pacman, yum, rpm - ディストリビューションに応じたパッケージマネージャー
sudo - Linuxで権限を昇格するコマンド
cal - コマンドラインカレンダーを表示
alias - 頻繁に使用するコマンドのカスタムショートカットを作成
dd - 主にブータブルUSBスティックを作成するために使用
whereis - コマンドのバイナリ、ソース、マニュアルページを検索
whatis - コマンドの使用目的を見つける
top - システム使用量と共にライブでアクティブなプロセスを表示
useradd and usermod - 新しいユーザーを追加するか既存のユーザーデータを変更
passwd - 既存のユーザーのパスワードを作成または更新
