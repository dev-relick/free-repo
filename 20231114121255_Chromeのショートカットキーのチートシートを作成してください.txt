Chromeのショートカットキーのチートシートを作成してください


Google Chromeは多くの便利なショートカットキーがあります。以下のチートシートは、よく使われるChromeのショートカットキーの一部をまとめたものです。

**注意**: ここで挙げるショートカットキーは、WindowsおよびLinuxのものです。Macユーザーは、`Ctrl`キーの代わりに`Command`キーを、`Alt`キーの代わりに`Option`キーを使います。

### タブとウィンドウの管理
| ショートカット | 説明 |
| --------------- | ----------- |
| `Ctrl` + `N` | 新しいウィンドウを開く |
| `Ctrl` + `Shift` + `N` | 新しいシークレットウインドウを開く |
| `Ctrl` + `T` | 新しいタブを開く |
| `Ctrl` + `Shift` + `T` | 閉じたタブを開き直す |
| `Ctrl` + `W` / `Ctrl` + `F4` | アクティブなタブを閉じる |
| `Ctrl` + `Tab` / `Ctrl` + `PgDn` | 次のタブに移動 |
| `Ctrl` + `Shift` + `Tab` / `Ctrl` + `PgUp` | 前のタブに移動 |
| `Ctrl` + `1` ～ `Ctrl` + `8` | タブバーの特定の位置にあるタブに移動 |
| `Ctrl` + `9` | タブバーの最後のタブに移動 |

### ウェブページのナビゲーション
| ショートカット | 説明 |
| --------------- | ----------- |
| `Alt` + `D` / `Ctrl` + `L` | アドレスバーにフォーカスする |
| `F5` / `Ctrl` + `R` | ページをリロードする |
| `Ctrl` + `F5` / `Shift` + `F5` | キャッシュを無視してページをリロードする |
| `Esc` | ロード中のページを停止する |
| `Alt` + `Left Arrow` / `Backspace` | 前のページに戻る |
| `Alt` + `Right Arrow` / `Shift` + `Backspace` | 次のページに進む |

### ウェブページの検索とナビゲーション
| ショートカット | 説明 |
| --------------- | ----------- |
| `Ctrl` + `F` | ページ内検索 |
| `Ctrl` + `G` / `F3` | ページ内検索で次を検索 |
| `Ctrl` + `Shift` + `G` / `Shift` + `F3` | ページ内検索で前を検索 |
| `Ctrl` + `H` | 履歴を表示する |
| `Ctrl` + `J` | ダウンロードを表示する |
| `Ctrl` + `D` | 現在のページをブックマークする |
| `Ctrl` + `Shift` + `O` | ブックマークマネージャーを開く |

### テキスト操作
| ショートカット | 説明 |
| --------------- | ----------- |
| `Ctrl` + `C` | 選択したテキストをコピー |
| `Ctrl` + `X` | 選択したテキストをカット |
| `Ctrl` + `V` | クリップボードの内容を貼り付け |
| `Ctrl` + `A` | 全てを選択 |

### Chrome機能
| ショートカット | 説明 |
| --------------- | ----------- |
| `Ctrl` + `Shift` + `B` | ブックマークバーの表示/非表示を切り替える |
| `Ctrl` + `Shift` + `J` | 開発者ツールを開く |
| `Ctrl` + `Shift` + `Delete` | ブラウジングデータをクリアする画面を開く |
| `Ctrl` + `P` | 印刷ダイアログを開く |
| `Ctrl` + `S` | ページを保存する |
| `F12` | 開発者ツールを開く |

これらのショートカットキーを覚えておくと、日々のブラウジングがより効率的になります。全てのショートカットを使いこなすのは大変かもしれませんが、よく使う機能に関するものから徐々に身につけていくのがおすすめです。
