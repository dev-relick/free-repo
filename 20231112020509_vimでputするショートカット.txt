vimでputするショートカット


Vimでは、put コマンド (`:put`) は、レジスタの内容をバッファに挿入する際に使われます。対話モードで使用することもできますが、キーボードショートカットを使用してノーマルモード（通常モード）で簡単に実行することができます。

特定のレジスタからペーストするには、まずそのレジスタを選択します。例えば、デフォルトレジスタからのペーストを行いたい場合は以下のキーシーケンスを使用します：

```
" + p
```

ここで、`"` はレジスタを指定するために使われ、`p` はカーソルの後ろにレジスタの内容をペーストします。

レジスタを指定せずにデフォルトレジスタからの内容を現在のカーソル位置の後ろにペーストするには、単に `p` キーを押します。前にペーストしたい場合は、`P`（大文字のP）を使用します。

また、例えば `"a` として名前付きレジスタを使って内容をペーストすることもできます（この例では、`a` レジスタを参照します）。これは以下のように行います：

```
"a + p （レジスタ a の内容をカーソルの後ろにペースト）
"a + P （レジスタ a の内容をカーソルの前にペースト）
```

クリップボードからペーストしたい場合、多くのシステムではレジスタ `+` を使います：

```
"+p （システムクリップボードの内容をカーソルの後ろにペースト）
"+P （システムクリップボードの内容をカーソルの前にペースト）
```

これらのコマンドはVimのデフォルトの挙動であり、これを使用するためにはVimがクリップボードをサポートしている必要があります。`vim --version` を実行して `+clipboard` が含まれているかどうかを確認できます。

Vimではカスタマイズが可能であるため、もしputに関する特定の動作を望む場合は、それをマップすることもできます。例えば、特定のキーを押すと自動でレジスタからペーストするようなキーマッピングを設定することができます。
